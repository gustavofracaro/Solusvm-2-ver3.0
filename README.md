## Instalação (Português)
1 - Baixe ou Clone os arquivos deste repositório

2 - Os arquivos baixados ou clonados devem estar dentro da seguinte pasta em seu WHMCS  /modules/servers/
3 - Crie seu servidor no WHMCS e escolha como módulo "SolusVM 3.0 VPS"

4 - Em Ip ou Hostname insira o do seu servidor Solusvm 2

5 - Em senha, insira sua chave API Key gerada no servidor Solusvm 2

6 - Teste a conexão e avance repetindo na proxima tela o Ip do seu servidor.

7 - Crie o seu produto de servidor, escolha o módulo "SolusVM 3.0 VPS" e aguarde as opções de configuração carregar.

8 - Teste realizando a compra com um cliente teste. Um plano de pacote deve ser criado antes no Solusvm.

9 - Caso as opções não apareçam e solicite o numero de licença é necessário aquiquirir uma em cloudatlantico@gmail.com

10 - Se quiser testar solicite uma licença trial por 5 dias.

11 - Insira o numero de licença fornecido e as opções serão liberadas.

12 - Em opções configuraveis, crie opções para Ipv4 Adicional, Ipv6 Adicional, Disco Adicional, Backup, Sistema Operacional

## Installation (English)
1 - Download or clone the files from this repository.

2 - The downloaded or cloned files must be located in the following folder in your WHMCS: /modules/servers/

3 - Create your server in WHMCS and choose "SolusVM 3.0 VPS" as the module.

4 - In IP or Hostname, enter your Solusvm 2 server's IP address.

5 - In password, enter your API Key generated on the Solusvm 2 server.

6 - Test the connection and proceed by repeating your server's IP address on the next screen.

7 - Create your server product, choose the "SolusVM 3.0 VPS" module and wait for the configuration options to load.

8 - Test by making the purchase with a test client. A package plan must be created beforehand in Solusvm.

9 - If the options do not appear and it requests a license number, you need to purchase one at cloudatlantico@gmail.com.

10 - If you want to test, request a 5-day trial license.

11 - Enter the provided license number and the options will be unlocked.

12 - In configurable options, create options for Additional IPv4, Additional IPv6, Additional Disk, Backup, Operating System.
