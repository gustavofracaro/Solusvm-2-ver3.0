Variáveis do módulo para modelos de e-mail (WHMCS)

Lista de variáveis normalmente disponíveis em templates de e-mail de Produto/Serviço do WHMCS para serviços do módulo SolusVM2VPS.

1) Variáveis padrão de serviço (mais utilizadas)

Descrição Variável
ID do serviço{$service_id}
Nome do produto{$service_product_name}
Domínio/hostname do serviço{$service_domain}
Usuário do serviço{$service_username}
Senha do serviço{$service_password}
IP principal do servidor{$service_server_ip}
Hostname do servidor{$service_server_hostname}
Nome do servidor no WHMCS{$service_server_name}
1º DNS{$service_server_ns1}
2º DNS{$service_server_ns2}

2) Campos customizados do produto (custom fields)

Campo do módulo/produto Variável no template
SolusVM server ID (vserverid){$service_custom_field_solusvm_server_id} ou {$service_custom_field_vserverid}
Restore backup id{$service_custom_field_restore_backup_id}
SSH Key{$service_custom_field_ssh_key}

3) Configurações do módulo (Config Options do produto)

O módulo define as opções abaixo no produto (plan, location, os_image, application, etc.). Em templates do WHMCS elas podem ser usadas por nome (quando suportado) ou por índice legado.

Config Option Variável sugerida Índice legado
plan{$service_config_option_plan}{$service_config_option_1}
location{$service_config_option_location}{$service_config_option_2}
os_image{$service_config_option_os_image}{$service_config_option_3}
application{$service_config_option_application}{$service_config_option_4}
user_data{$service_config_option_user_data}{$service_config_option_5}
backup_enabled{$service_config_option_backup_enabled}{$service_config_option_6}
role{$service_config_option_role}{$service_config_option_7}
limit_group{$service_config_option_limit_group}{$service_config_option_8}
additional_disk_offer{$service_config_option_additional_disk_offer}{$service_config_option_9}
snapshot_enabled{$service_config_option_snapshot_enabled}{$service_config_option_10}
360monitoring_enabled{$service_config_option_360monitoring_enabled}{$service_config_option_11}
ssh_key{$service_config_option_ssh_key}{$service_config_option_12}
disaster_recovery_enabled{$service_config_option_disaster_recovery_enabled}{$service_config_option_13}
Observação importante: a disponibilidade exata dos placeholders pode variar conforme o tipo do template de e-mail no WHMCS e a versão instalada. Em caso de dúvida, use um e-mail de teste no WHMCS para validar cada variável.
